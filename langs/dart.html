Dart



Dart는 구글이 개발한 프로그래밍 언어입니다.
C의 문법과 유사하지만, 객체지향, 클래스 기반, 단일 상속 등의 특징을 가지고 있으며,
정적 타입 검사와 동적 타입 검사를 함께 지원합니다.
AngularDart, Flutter 등의 UI 프레임워크에서 사용되며, 최근 모바일 애플리케이션 개발에 활발하게 이용됩니다.
JavaScript와 같이 클라이언트 측에서 사용할 수도 있지만, 서버 측에서도 사용 가능하여 Node.js와 함께 사용할 수 있습니다.


해당 그래프는 공식적인 자료가 아닙니다. 단순 참고용으로만 사용해주시길 바랍니다.
정적/동적 타이핑의 경우 음수일 경우 동적 타이핑, 양수일 경우 정적 타이핑을 지원하는 지표로 나타내었습니다.
Dart가 사용된소프트웨어:
- Flutter
- Fuchsia OS
- Google Stadia
- Soundtrap
- Google Ads
