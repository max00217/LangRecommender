JavaScript



JavaScript는 웹브라우저 상에서 동작하는 스크립트 언어이며, 최근에는 Node.js를 통해 서버 측에서도 사용됩니다.
클라이언트에서 동작하는 스크립트 언어로 유저 인터페이스 개발에서 활용되며,
최근에는 Node.js와 함께 프론트엔드 뿐만 아니라 백엔드 어플리케이션 개발에도 이용됩니다.
동적 타이핑을 지원하여 유연한 코딩이 가능하며, jQuery, React, AngularJS 등 많은 프레임워크와 라이브러리가 제공됩니다.
TypeScript라는 JavaScript의 단점을 보완한 슈퍼셋 언어도 개발되었습니다.


해당 그래프는 공식적인 자료가 아닙니다. 단순 참고용으로만 사용해주시길 바랍니다.
정적/동적 타이핑의 경우 음수일 경우 동적 타이핑, 양수일 경우 정적 타이핑을 지원하는 지표로 나타내었습니다.
