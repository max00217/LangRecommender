Kotlin



Kotlin은 JVM(Java Virtual Machine) 기반의 프로그래밍 언어로, Jetbrains는 Java의 대체 언어로 Kotlin의 사용을 권장합니다.
안드로이드 개발을 비롯한 여러 플랫폼에서 사용이 가능합니다.
타입 안정성을 보장하며, null-safe한 것이 특징입니다.
함수형 프로그래밍과 객체지향 프로그래밍 방식을 모두 지원하는 멀티 패러다임 언어입니다.
Java 대비 간결한 문법과 코루틴을 통해 생산성을 높이고, 코드의 가독성을 높일 수 있습니다.


해당 그래프는 공식적인 자료가 아닙니다. 단순 참고용으로만 사용해주시길 바랍니다.
정적/동적 타이핑의 경우 음수일 경우 동적 타이핑, 양수일 경우 정적 타이핑을 지원하는 지표로 나타내었습니다.
Kotlin이 사용된 소프트웨어:
- Threads
- 카카오톡 메시징 서버
- Gradle
- Pinterest 앱
- Android Studio
