PHP



PHP는 서버 측에서 동작하는 스크립트 언어입니다.
동적인 웹페이지 생성, 웹 기반의 소프트웨어 개발 등 다양한 목적으로 사용이 가능합니다.
C, Java 등과 같은 언어들로 작성된 확장 기능을 모아둔 PECL 라이브러리나 PEAR 및 Composer와 같은 라이브러리 관리 프로그램을 통해 높은 확장성을 보유하고 있습니다.
프로그램의 일부분을 HTML 문서에 직접 구현할 수 있습니다.
안정성이 높고 빠른 개발이 가능하지만, 대용량 트래픽 서비스에는 취약한 면도 있습니다.


해당 그래프는 공식적인 자료가 아닙니다. 단순 참고용으로만 사용해주시길 바랍니다.
정적/동적 타이핑의 경우 음수일 경우 동적 타이핑, 양수일 경우 정적 타이핑을 지원하는 지표로 나타내었습니다.
PHP가 사용된 소프트웨어:
- 네이비즘
- Laravel
- 리로스쿨
- Gnuboard
- Pixiv
